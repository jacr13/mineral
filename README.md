# mineral

A minimal robot learning library that aggregates reliable implementations for:

- PPO ([`minimal-stable-PPO`](https://github.com/ToruOwO/minimal-stable-PPO), [`rl_games`](https://github.com/Denys88/rl_games)) #rl

- DDPG, TD3 ([`pql`](https://github.com/Improbable-AI/pql), [`drqv2`](https://github.com/facebookresearch/drqv2), [`TD3`](https://github.com/sfujim/TD3)) #rl

- SAC ([`pql`](https://github.com/Improbable-AI/pql), [`pytorch-sac-ae`](https://github.com/denisyarats/pytorch_sac_ae)) #rl

- SHAC, BPTT/APG ([`DiffRL`](https://github.com/NVlabs/DiffRL)) #rl, #diffsim

- BC #il

- DAPG ([`maniskill2-learn`](https://github.com/haosulab/ManiSkill2-Learn)) #il, #rl, #off2on
